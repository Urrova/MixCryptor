Help:\n\n\nEncrypt: Here you put the root of the file to encrypt and the key file and the root for save the encrypted file.\n\nDecrypt: Here you put the root of the file to decrypt and the key file and the root for save the decrypted file.\n\nGenerate Key: Here you make the key to encrypt/decrypt. You put the root to save the key file the size of the keys the format and the characters quantity.
Ayuda:\n\n\nCifrar: aqui se coloca la raiz del archivo a cifrar y el archivo de clave y la raiz para guardar el archivo cifrado.\n\nDescifrar: Aqui se coloca la raiz del archivo para descifrar y el archivo de clave y la raiz para guardar el archivo descifrado.\n\nGenerar clave: aqui crea la clave para cifrar / descifrar. Usted coloca la raiz para guardar el archivo de clave el tamaño de las claves el formato y la cantidad de caracteres.
Socorro:\n\n\nCriptografar: aqui voce coloca a raiz do arquivo para criptografar e o arquivo de chave e a raiz para salvar o arquivo criptografado.\n\nDescriptografar: Aqui voce coloca a raiz do arquivo para descriptografar e o arquivo-chave e a raiz para salvar o arquivo descriptografado.\n\nGerar Chave: Aqui voce faz a chave para criptografar / descriptografar. Voce coloca a raiz para salvar o arquivo-chave o tamanho das chaves o formato e a quantidade de caracteres.
